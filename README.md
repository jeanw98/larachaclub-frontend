# Amici Frontend

Mobile-first web app to pin photos on a world map with friends.

## Setup

```bash
nvm use 24
npm install
npm run dev
```

Runs on **http://localhost:5173** (proxies API to backend on port 3001).

Make sure the backend is running first.

## Features

- 📍 Pin photos anywhere on the world map
- 😂 React with funny, awful, scary, love, wow, meh
- ⭐ Rate pins with comments (1-5 stars)
- 🏆 Leaderboard with auto-generated cool names
- 🔥 Heatmap by density or reaction type
- 📱 Mobile-first, responsive desktop layout
